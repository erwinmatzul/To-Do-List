# 📝 To-Do List

 Una app simple y elegante para gestionar tus tareas diarias. Organiza tu día, mantente enfocado y ¡haz que cada tarea cuente! 🚀

## 🌟 Funcionalidades

-  Agrega nuevas tareas fácilmente  
-  Marca tareas como completadas  
-  Elimina tareas cuando ya no las necesites
